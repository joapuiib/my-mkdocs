    Icons

    Devicons

    Icons from devicons are available.

    You can use the following syntax to include an icon:

    Example with Java

    Syntax Icon
    :devicons-java-original:
    :devicons-java-original-wordmark:
    :devicons-java-plain:
    :devicons-java-plain-wordmark:

    Example with Python

    Syntax Icon
    :devicons-python-original:
    :devicons-python-original-wordmark:
    :devicons-python-plain:
    :devicons-python-plain-wordmark:

    Built-in icons

    A complete list can be found at https://squidfunk.github.io/mkdocs-material/reference/icons-emojis/#search

    Page icons

    In the frontmatter: 

    icon: devicons/java-plain
